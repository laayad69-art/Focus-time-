## 🧠 About The Project

**Focus Engineer Timer** هو نظام متكامل لإدارة الوقت بأسلوب Pomodoro  
مصمم مخصوص لطلاب الهندسة والناس اللي عايزة تركيز حقيقي 🔥

بيجمع بين:
- ⏱ إدارة الوقت  
- 📊 تتبع الأداء  
- 💬 التحفيز  
- 🤲 الجانب الروحي

## ✨ Features

### ⏱ Smart Timer
- دائرة متحركة تقل مع الوقت  
- تأثير Neon Glow  
- 3 Modes:
  - 🎯 Focus  
  - ☕ Short Break  
  - 🛌 Long Break  
- 📉 Progress Bar  

---

### ⚙️ Customization
- تعديل وقت الجلسات  
- تحكم كامل:
  - ▶️ Start  
  - ⏸ Pause  
  - 🔄 Reset  
  - ⏭ Skip  

---

### ✅ Task Manager
- إضافة المواد  
- تحديد المهام بعد الانتهاء ✔️  
- تنظيم المذاكرة بسهولة  

---

### 📊 Statistics
- عدد الجلسات  
- إجمالي وقت التركيز  
- 🔥 Streak (أيام متتالية)  

---

### 💾 Weekly Tracking
- تسجيل تلقائي للجلسات  
- 📅 مخطط 7 أيام  
- اليوم الحالي مميز  

---

### 💬 Motivation System
- تحديث كل 30 ثانية  
- بعد كل جلسة  

💡 أمثلة:
> "النجاح في الهندسة = 1% إلهام + 99% حسابات صح"  
> "فهم الأساسيات هو القوة الحقيقية"  
> ﴿وَقُل رَّبِّ زِدْنِي عِلْمًا﴾  

---

### 🔔 Sound Alerts
- تنبيه عند انتهاء الجلسة  
- إمكانية تشغيل/إيقاف
